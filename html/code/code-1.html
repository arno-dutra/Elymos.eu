Elymos.eu

J’ai développé ce site dans l’objectif de tenir un journal professionnel, une sorte de CV dynamique qui permettrait de refleter mon approche multidisciplianire.

Ce site a par ailleurs pour moi été l’occasion de faire mes premiers pas en HTML, CSS et JS (dont JQuery). Je l’ai architecturé pour qu’il se construise dynamiquement côté client en s’appuyant sur deux fichiers JSON décrivant les fils et les posts.

L’ensemble du code peut être retrouvé en OpenSource sur mon github.
